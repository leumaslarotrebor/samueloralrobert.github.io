PROJECT · 01
DocuMind
⎆ VIEW ON GITHUB
RAG-based AI Document Assistant — an end-to-end document intelligence pipeline that lets users semantically query any document corpus with LLM-powered contextual answers.
Document Upload → Chunk + Embed → FAISS Index → LLM Synthesis
FastAPI LangChain FAISS OpenAI API Python
Architecture Notes
Recursive text splitter with configurable chunk size and overlap
FAISS flat L2 index with cosine similarity reranking
Prompt templates with context injection and hallucination guards
Async FastAPI endpoints for streaming LLM responses

SamLang
⎆ VIEW ON GITHUB

Architecture Notes

AST visitor pattern for semantic analysis
Custom symbol table with scope resolution
Bytecode generation with stack-based VM
